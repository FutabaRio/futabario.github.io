Welcome To My Blog

发表于 2019-11-14

样子就先是这么个样子吧，等自己Django学好成之后自己会重写这个博客网站的
总之网站总算是先弄出来了，Hexo+Github。域名是阿里云首年9块（椎名真白天下第一）下面说一下具体的步骤

过程

需要的环境：Node.js ,Git,github

GIt

下载安装（略），需要设置环境变量。

最终会生成一个文件在用户目录下，打开用户目录，找到.ssh\id_rsa.pub文件，记事本打开并复制里面的内容，打开你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key：

如果提示Are you sure you want to continue connecting (yes/no)?，输入yes，然后会看到：

! You’ve successfully authenticated, but GitHub does not provide shell access你的ssh就已经配置好了

Hexo的安装

安装

在电脑的某个地方新建一个hexo文件夹（名字可以自定义）这里用d:\hexo举例，这里是用于存放你代码的地方.右键打开git bash

打开浏览器，输入localhost:4000，就可以在本地浏览你的blog

选择主题

打开博客根目录下的_config.yml文件找到theme行可以修改主题
主题文件在根目录themes文件夹中。例如(next主题在github可以下载)

配置github项目连接

打开博客根目录下_config.yml文件找到deploy行添加

repo地址:直接把你的用户名和项目名替换
例如: futabario.github.io
/futabario/futabario.github.io.git （前是仓库名后是项目名）

使用git部署

域名绑定

  1. 域名购买在阿里云上面很容易就完成，域名需要实名验证

  2. 设置域名解析，点击域名列表找到域名，点击解析选项，填入你的博客解析的ip地址（在cmd下ping一下域名）

  3. 在博客根目录的public文件夹中创建一个CNAME.txt文件 里面写入你购买的域名，写入后把文件后缀.txt去掉，修改为普通文件（不用协议）例如：shiinamashiro.top

  4. 在github中setting里，找到custom domain 填入域名 例如：shiinamashiro.top

  5. 再重新上传一次 （hexo g ，hexo d）
